**AI₂O₃ — Domain-Specific Language for Orthogonal, Ordinal, and Observable AI Reasoning**

## Core Features (from BNF v1.6.5 "The Camera & Lens Canon")

- Zero-Hallucination Syntax: Sigil-heavy, LL(1)-compatible grammar eliminates prediction uncertainty  
- DOD Native: No objects; direct array manipulation, pre-allocated schemas  
- Lens/Scene Model: `[> [` (Wide/Base), `|>[` (Normal), `]> [` (Tele/Overlay) for layered observable execution  
- Automatic Iteration & O(N) Enforcement: Nested loops forbidden; iteration via diverse sources only  
- QA Chain & I/O Gates: Deterministic flow (`?`, `|?`, `|!`) and observable I/O (`<`, `>`, `x`, `_`, `?`)  
- Strict Physicality: Range-based indexing, no zero-offset errors, explicit mutation control
